CleverStyle CMS

Проста, швидка і масштабована система керування контентом.

Найкраще задовольняє вимоги розробників, які створюють сайти з унікальною функціональністю.

Вцілому система має всі необхідні абстракції для швидкої розробки, які працюють відразу після встановлення, і не потребують сцеціального налаштування через зважені налаштування за замовчуванням.

Насправді це CMF, але розвивається в напрямку CMS щоб бути більш зручною.

PickMeUp jQuery Datepicker плагін

Насправді простий і функціональний datepicker, який легко налаштовувати

jQuery datepicker з простим інтерфейсом та простим налаштуванням зовнішнього вигляду.

Може бути використаний з полями вводу, або просто на сторінці. Підтримує вибір однієї дати, кількох та інтервалу.

SASS стилі з параметрами дозволяють легко підігнати datepicker під ваш проект.

Useful PHP Functions

Набір системно-незалежних функцій з CleverStyle CMS

Це лише набір корисних функцій які більшою чи меншою мірою тривіальні. Більшість з них були написані для CleverStyle CMS, але, сподіваюсь, вони можуть бути корисними ще комусь.

Ціль UPF це поширення DRY принципу та уникання тривіальних надоїдливих блоків коду на користь того, що насправді має значення.

PHP Markdown Next

Парсер що базується на PHP Markdown Extra та PHP Markdown Extra Extended

Парсер для Markdown розмітки з різними корисними розширеннями.

Рефакторинг старого коду, додані корисні можливості, які не доступні ні в PHP Markdown, ні в PHP Markdown Extra, може бути встановлений з використанням composer.

CleverStyle PhpStorm theme

Темна тема для PhpStorm/WebStorm

Є ненульова вірогідність, що якщо ви читаєте цей текст — то ви програмуєте щось з використанням PHP, JavaScript, CSS і так далі. Для багатьох людей читати світлий текст на темному фоні набагато легше, ніж темний текст на світлому фоні.

Це хороша темна тема, яку я використовую багато годин щодня з PhpStorm IDE, сподіваюсь, вона вам теж сподобається:)

BananaHTML

BananaHTML — один клас, який робить генерацію HTML простішою

Втомились від змішаних шаблонів з HTML/PHP, не подобається шаблонізатори, синтаксис яких є ще однією новою мовою розмітки?

Ось рішення: генеруйте HTML з PHP напряму за допомогою добре відомих CSS-селекторів. Постий, природній синтаксис, хороша швидкість і ніякої потреби кешувати шаблони - вони і є звичайним PHP файлами.

CleverStyle Tester

CleverStyle Tester це простий інструмент для тестування ваших PHP додатків

Якщо вам не подобається великі багатофункціональні фреймворки для тестування — цей інструмент абсолютно протилежний.

Це дуже простий інструмент, який дозволяє вам тестувати додатки через браузер або командний рядок, і робить це інтерактивно, тобто ви можете бачити прогрес в реальному часі.
