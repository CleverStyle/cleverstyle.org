CleverStyle CMS

Простая, быстрая и масштабируемая система управления контентом.

Лучше всего удовлетворяет требования разработчиков, которые создают сайты с уникальной функциональностью.

В целом система имеет все необходимые абстракции для быстрой разработки, которые работают сразу после установки и не требуют дополнительной настройки брагодаря взвешенным настройкам по умолчанию.

На самом деле это CMF, но развивается в направлении CMS чтобы быть более удобной.

PickMeUp jQuery Datepicker плагін

На самом деле простой и функциональный datepicker, который легко настраивать

jQuery datepicker с простым интерфейсом и простой настройкой внешнего вида.

Может быть использован с полями ввода, или просто на странице. Поддерживает выбор одной даты, нескольких и интервала.

SASS стили с параметрами позволяют легко подогнать datepicker под ваш проект.

Useful PHP Functions

Набор системно-независимых функций с CleverStyle CMS

Это лишь набор полезных функций в большей или меньшей степени тривиальных. Большинство из них были написаны для CleverStyle CMS, но, надеюсь, они могут быть полезными еще кому-то.

Цель UPF это распространение DRY принципа и избежание тривиальных надоедливых блоков кода в пользу того, что на самом деле имеет значение.

PHP Markdown Next

Парсер который базируется на PHP Markdown Extra и PHP Markdown Extra Extended

Парсер для Markdown рразметки с разными расширениями.

Рефакторинг старого кода, добавлены полезные возможности, которые не доступны ни в PHP Markdown, ни в PHP Markdown Extra, может быть установлен с использованием composer.

CleverStyle PhpStorm theme

Темная тема для PhpStorm/WebStorm

Есть ненулевая вероятность, что если вы читаете этот текст - то вы программируете что-то с использованием PHP, JavaScript, CSS и так далее. Для многих людей читать светлый текст на темном фоне гораздо легче, чем темный текст на светлом фоне.

BananaHTML

BananaHTML — один класс, который делает генерацию HTML проще

Устали от смешанных шаблонов с HTML/PHP, не нравятся шаблонизаторы, синтаксис которых является еще одним новым языком разметки?

Вот решение: создавайте HTML с PHP непосредственно с помощью хорошо известных CSS-селекторов. Постой, естественный синтаксис, хорошая скорость и никакой необходимости кэшировать шаблоны - они и есть обычным PHP файлами.

CleverStyle Tester

CleverStyle Tester это простой инструмент для тестирования ваших PHP приложений

Если вам не нравится большие многофункциональные фреймворки для тестирования - этот инструмент совершенно противоположный.

Это очень простой инструмент, который позволяет вам тестировать приложения через браузер или командную строку, и делает это интерактивно, то есть вы можете видеть прогресс в реальном времени.
