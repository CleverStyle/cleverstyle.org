CleverStyle CMS

CleverStyle CMS це проста, масштабована, і швидка система керування контентом.

Поточна версія Alpha.

Ядно системи містить тільки базову функціональність, тому більше схоже на CMF (фреймворк керування контентом) ніж на CMS.

Але паралельно з ядром розробляються деякі компоненти:

Модулі

  • Blogs (проста блогова функціоналність)
  • Comments (додає функціональність коментарів в інші модулі)
  • Cron (простий графічний інтерфейс ядля crontab, запланованих задач)
  • Disqus (інтеграція сервісу коментування Disqus, додає функціональність коментарів в інші модулі)
  • Feedback (простий модуль зворотнього зв'язку, відправляє повідомлення на email адміністратора)
  • HybridAuth (інтеграція бібліотеки HybridAuth для інтеграції з соціальними мережами та іншими сервісами)
  • OAuth2 (забезпечує реалізацію протоколу авторизації OAuth 2 (серверна сторона))
  • Photo gallery new (простий модуль фотогалереї, що базується на компонентах Fotorama, Plupload і SimpleImage)
  • Plupload (інтеграція Plupload для завантаження файлів, додає функціональність завантаження файлів в інші модулі)
  • Static pages (дозволяє створювати прості статичні сторінки, наприклад Про Нас, чи сторінки без інтерфейсу, наприклад для підтвердження володіння сайтом)

Плагіни

  • Admin default theme (примусово встановить тему CleverStyle в адмініструванні, що буває корисно, так як не потрібно адаптувати тему для адмін-панелі)
  • Fotorama new (інтеграція jQuery галереї Fotorama в CleverStyle CMS)
  • SimpleImage new (інтеграція класу SimpleImage в CleverStyle CMS)
  • TinyMCE (інтеграція TinyMCE WYSIWYG Редактора для забезпечення простого редагування контенту)

Основні можливості

  • Підтримка компонентів
    • Підтримка модулів (для відображення основного вмісту сторінки)
    • Підтримка плагінів (завантажуються на кожній сторінці і забезпечують додаткову функціональність)
    • Підтримка блоків (розміщуються насколо сторінки для відображення додаткової інформації)
  • Підтримка зрозумілих людині адрес сторінок
  • Підтримка груп та типів користувачів
  • Підтримка прав та контролю доступу
  • Повна багатомовна підтримка (як інтерфейсу, так і контенту)
    • Підтримка автоматичнного перекладу контенту
  • Підтримка мініфікації і компресії CSS та JavaScript
  • Підтримка дзеркал сайту
    • Підтримка дзеркал доменів
    • Підтримка дзеркал фізичних серверів для кожного домену
  • Пітримка тем та кольорових схем
  • Пітримка кількох баз даних
    • Підтримка кількох дзеркал баз даних
  • Пітримка кількох сховищ файлів
  • Підтримка системного кешування
  • Фільтр та обмеження по IP
  • Підтримка зовнішнього API сайту
  • Підтримка IPv6

Вимоги

  • Unix-подібна операційна система
  • Веб-сервер Apache2 або Nginx
  • PHP 5.4+
    • Бібліотека mcrypt (рекомендується для шифрування)
    • Бібліотека cUrl (рекомендується для автоматичних перекладів)
    • APC (Alternative PHP cache) модуль (рекомендується для прискорення системи з PHP 5.4)
  • MySQL сервер баз данных (MariaDB також буде працювати)
    • Система може бути розширена для підтримки інших серверів баз данных
